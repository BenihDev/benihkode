Gas Flow Calculator is a mobile engineering utility built for field workflows where precision, clarity, and offline-friendly ergonomics matter.

## What It Does

- Calculates gas flow using ISO 5167-oriented engineering inputs
- Supports precision steppers for small field adjustments
- Keeps unit switching close to each input for faster metric and imperial conversion
- Uses a floating pill navigation pattern to preserve calculator screen space
- Provides a streamlined support path for bug reports and app feedback

## Field-First Design

The app is designed around practical field constraints: small screens, one-handed use, gloves, and the need to make accurate changes without fighting the keyboard.
