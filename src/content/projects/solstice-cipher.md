A free, offline puzzle game for Android, built in Godot 4 for the June Solstice Game Jam. Solstice Cipher teaches its optical mechanics through an unobtrusive, context-aware tutorial — dynamic tweens and signals guide players without breaking immersion.

Get it on [Google Play](https://play.google.com/store/apps/details?id=com.fanioz.solsticecipher), or try the web build on [GitHub Pages](https://fanioz.github.io/solstice-cipher/) / [Vercel](https://solsticechiper.vercel.app/). Read the dev journal on [building the tutorial UI](/blog/solstice-chiper).
